Pengantar JavaScript

Sejarah JavaScript

JavaScript pertama kali dikembangkan oleh Brendan Eich pada tahun 1995 saat bekerja di Netscape Communications. Awalnya dinamakan Mocha, kemudian diganti menjadi LiveScript, dan akhirnya dinamakan JavaScript untuk memanfaatkan popularitas bahasa Java. Meski namanya mengandung kata "Java," JavaScript berbeda dari Java baik dalam sintaks maupun penggunaannya.

JavaScript menjadi bagian dari browser Netscape Navigator dan dengan cepat menjadi standar de facto untuk pemrograman web. Pada tahun 1997, JavaScript distandarkan oleh ECMA International dengan nama ECMAScript. Standar ini terus berkembang dengan versi terbaru dirilis setiap tahun.

Apa Itu JavaScript?

JavaScript adalah bahasa pemrograman tingkat tinggi yang digunakan terutama untuk pengembangan web. Dengan JavaScript, pengembang dapat membuat halaman web yang interaktif, seperti menambahkan animasi, memvalidasi formulir, atau membuat aplikasi web kompleks.

JavaScript berjalan di browser web dan dapat digunakan di sisi klien (client-side) maupun di sisi server (server-side) dengan menggunakan platform seperti Node.js.

Fitur Utama JavaScript

  • Dinamis: JavaScript adalah bahasa pemrograman yang fleksibel dan tidak membutuhkan deklarasi tipe data.
  • Interaktif: Membantu membuat pengalaman pengguna yang lebih baik melalui elemen interaktif pada halaman web.
  • Berbasis Prototipe: JavaScript mendukung pewarisan berbasis prototipe daripada pewarisan berbasis kelas seperti di banyak bahasa lainnya.
  • Multiplatform: Dapat dijalankan di berbagai platform, baik browser maupun server.

Contoh Dasar JavaScript

Berikut adalah contoh program JavaScript sederhana:

// Menampilkan pesan di konsol
console.log("Hello, JavaScript!");

// Menambahkan dua angka
let a = 5;
let b = 3;
let sum = a + b;
console.log("Hasil penjumlahan: " + sum);

Penutup

JavaScript adalah salah satu bahasa pemrograman yang paling penting dalam pengembangan web modern. Dengan memahami JavaScript, Anda dapat membuat aplikasi web yang interaktif dan dinamis. Mulailah dengan dasar-dasarnya, dan Anda akan siap untuk mengeksplorasi fitur-fitur yang lebih canggih!
